Code Information

Morse Code

Morse code is a method used in telecommunication to encode text characters as sequences of two different signal durations, called dots and dashes. It is named after Samuel Morse, one of the inventors of the telegraph.

To encode text into Morse code, each character is represented by a series of dots (.) and dashes (-). For example, the letter 'A' is represented as ".-", and the letter 'B' is represented as "-...".

Binary

Binary code is a coding system using the binary digits 0 and 1 to represent characters or instructions in a computer or digital device. It is the fundamental language of computers.

In binary code, each character is represented by a sequence of 0s and 1s. For example, the letter 'A' can be represented as "01000001".

ASCII

ASCII (American Standard Code for Information Interchange) is a character encoding standard that represents text characters as integers. It is widely used in computers and electronic communication.

Each ASCII character is represented by a unique 7-bit binary number. For example, the letter 'A' is represented as 65 in decimal or 01000001 in binary.

Base64

Base64 is a binary-to-text encoding scheme that allows binary data to be represented as ASCII characters. It is commonly used for transmitting data over text-based protocols.

In Base64, each set of three bytes is represented by four characters. For example, the string "Hello" can be represented as "SGVsbG8=" in Base64.

Caesar Cipher

The Caesar cipher is a simple substitution cipher where each letter in the plaintext is shifted a certain number of positions down or up the alphabet. It is named after Julius Caesar, who is said to have used this cipher.

In the Caesar cipher, a shift of 1 means 'A' becomes 'B', 'B' becomes 'C', and so on. For example, with a shift of 3, the plaintext "HELLO" is encrypted as "KHOOR".
